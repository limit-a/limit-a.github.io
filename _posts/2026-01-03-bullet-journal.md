### 📌 시스템 소개

**제작자**: Ryder Carroll

**목적**: 
- 과거의 행적을 보관(Track the past)
- 현재를 정리(Order the present)  
- 미래를 계획(Design the future)

**특징**:
- 아날로그 시스템 (노트와 펜만 있으면 됨)
- 무엇을 쏟아놓든 정리할 수 있을 정도로 유연함
- 방해되지 않을 정도로 충분히 빠름
- 맞춤형 노트가 아니어도 어떤 노트에서든 작동 가능

---

### 📖 1단계: Index (색인) 설정

**위치**: 노트의 첫 번째 빈 양면

**설정 방법**:
1. 첫 양면을 펼침
2. 상단에 "Index" 제목 작성
3. 이후 모든 항목의 페이지 번호를 여기에 기록

**역할**: 노트의 목차 역할, 나중에 필요한 내용을 빠르게 찾을 수 있게 함

---

### 📅 2단계: Future Log (미래 일지) 설정

**위치**: Index 다음의 빈 양면

**설정 방법**:
1. 양면에 "Future Log" 제목 작성
2. 페이지의 총 줄 수를 셈
3. 총 줄 수를 3으로 나눔
4. 자를 사용해 양면에 선을 그어 6개 칸 만들기
5. 각 칸에 월(月) 이름 작성
6. 페이지 번호를 적고 Index에 "Future Log"와 페이지 번호 기록

**용도**: 몇 달 후의 일정이나 할 일을 미리 적어두는 공간

---

### 📆 3단계: Monthly Log (월간 일지) 설정

**위치**: Future Log 다음의 빈 양면

**설정 방법**:

**왼쪽 페이지 - Monthly Calendar (월간 달력)**:
1. 페이지 상단에 월 이름 작성
2. 그달의 모든 날짜를 세로로 나열 (1, 2, 3...)
3. 각 날짜 옆에 요일의 첫 글자 작성 (월=M, 화=T...)
4. 이벤트나 약속을 해당 날짜에 기록

**오른쪽 페이지 - Monthly Task List (월간 할 일 목록)**:
1. 페이지 상단에 월 이름 작성
2. 이번 달에 해야 할 모든 일을 나열
3. 각 항목 앞에 • (점) Task Bullet 표시
4. 양쪽 페이지에 페이지 번호 작성
5. Index에 해당 월과 페이지 번호 기록

**역할**: 한 달 동안 해야 할 모든 것과 일정을 한눈에 파악

---

### 📝 4단계: Daily Log (일일 일지) 작성

**설정 방법**:
1. 날짜 작성 (예: 1월 3일)
2. 그날의 일, 이벤트, 메모를 적기 시작

**Rapid Logging (빠른 기록법)**:

모든 항목은 짧은 문장(불렛)으로 작성하며, 세 가지 유형으로 분류:

| 기호 | 이름 | 용도 | 예시 |
|------|------|------|------|
| **•** | Task Bullet | 해야 할 일 | • 보고서 작성 |
| **○** | Event Bullet | 이벤트/약속 | ○ 오후 3시 회의 |
| **-** | Note Bullet | 메모/아이디어 | - 좋은 아이디어 떠오름 |

**Signifier (기호 표현)**:
- **\*** (별표): 중요한 항목 표시
  - 예: **\* •** 긴급 보고서 제출
- 다른 특별한 의미를 불렛에 추가할 수 있음

**장점**: 정보를 빠르게 담고 정리할 수 있음

---

### 🔄 5단계: Migration (이동) - 매달 말 진행

**목적**: 바쁜 것과 생산적인 것의 차이를 만들기 위함

**진행 순서**:

1. **각 달의 마지막에 다음 Monthly Log 시작**

2. **현재 Daily Log 검토**:
   - 완료한 일: **X** 표시
   - 모든 남은 일을 하나씩 검토

3. **각 남은 일에 대해 자문**:
   > "이 일이 아직도 내 시간을 들일 만한 가치가 있나요?"

4. **가치 판단에 따른 처리**:

   **가치 없음** → 줄 긋기 (취소)
   
   **다음 달에 할 가치 있음** 
   - • (점)를 → (오른쪽 화살표)로 변경
   - 새로운 Monthly Log에 다시 작성
   
   **몇 달 후에 할 일**
   - • (점)를 ← (왼쪽 화살표)로 변경  
   - Future Log에 작성

**효과**: 
- 주의를 산만하게 만드는 불필요한 일 제거
- 정말 중요한 일에만 시간과 에너지 집중
- 의식적으로 우선순위 설정

---

### 📚 Collection (수집물) 만들기

**용도**: 특정 주제와 관련된 일과 메모를 한곳에 정리

**만드는 방법**:
1. 빈 페이지로 이동
2. 상단에 주제/제목 작성
3. 페이지 번호 매기기
4. 관련된 모든 메모와 할 일을 이곳에 작성
5. Index에 Collection 제목과 페이지 번호 기록

**활용 예시**:
- 쇼핑 목록
- 진행 중인 프로젝트
- 수업 노트
- 읽고 싶은 책 목록
- 여행 계획
- 운동 기록

**장점**: 흩어진 정보를 주제별로 체계적으로 정리

### ✅ 전체 흐름 요약

```
1. Index (색인) 설정
   ↓
2. Future Log (6개월) 설정
   ↓
3. Monthly Log (이번 달) 설정
   - 왼쪽: 달력
   - 오른쪽: 할 일 목록
   ↓
4. Daily Log 매일 작성
   - • Task
   - ○ Event  
   - - Note
   ↓
5. 매달 말 Migration
   - X: 완료
   - —: 취소
   - →: 다음 달로
   - ←: Future Log로
   ↓
6. 필요시 Collection 추가
```

### 💡 핵심 철학

**"바쁜 것과 생산적인 것의 차이"**

불렛저널은 단순히 할 일을 나열하는 게 아니라:
- 정기적으로 검토하며 **불필요한 것을 제거**하고
- **정말 중요한 일에만 집중**하게 만드는
- **의식적이고 의도적인** 생산성 시스템입니다
